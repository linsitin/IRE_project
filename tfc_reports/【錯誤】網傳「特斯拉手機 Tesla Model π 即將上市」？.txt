標題：【錯誤】網傳「特斯拉手機 Tesla Model π 即將上市」？
來源：https://tfc-taiwan.org.tw/fact-check-reports/migration-10305/
--------------------------------------------------

【報告將隨時更新 2024/2/17版】

網傳特斯拉手機「Tesla Model π」即將上市，查證結果如下：

一、經檢索特斯拉官方網站與社群平台，均無「Tesla Model π」或特斯拉手機的相關資訊。

網傳特斯拉手機「Tesla Model π」是源於一位設計師在2021年的發想，後續有許多影片創作者或設計師跟進設計。傳言出現的手機照片出自個人YouTube頻道創作，非特斯拉官方發布訊息。

二、手機專家表示，目前特斯拉推出手機只在傳聞階段，尚無可信的證據佐證。

三、經檢索《路透社》、《CNN》、《美聯社》、《BBC》、《法新社》等外媒，均無特斯拉將發布手機的相關報導。

四、特斯拉要出手機是流傳已久的訊息，《法新社》、印度查核組織《Factly》與《YouTurn》都曾闢謠。

網傳「特斯拉手機 Tesla Model π 即將上市」是網路創作的概念手機，並非特斯拉發布的官方訊息；因此，傳言為「錯誤」訊息。

【查核聲明】資訊若有更新，本報告亦會同步更新。

背景

社群平台近期流傳一組手機照片，宣稱「特斯拉手機即將上市」。照片中的手機有四個鏡頭，手機背殼有特斯拉的LOGO，而手機盒上則寫著「Tesla Model π」。

傳言有多種版本，包括影片、圖片形式，而不同的傳言對此手機的規格也有不同描述，有的傳言稱特斯拉將上市的手機無流量限制；也有傳言稱特斯拉手機的電池將由太陽能供電，用戶可在日光下為特斯拉智慧型手機充電。

查核

爭議點、網傳「特斯拉手機 Tesla Model π 即將上市」，是否屬實？

特斯拉官方是否曾發布手機「Tesla Model π」資訊？

（一）經檢視特斯拉官方網站、官方推特、官方YouTube頻道等，與特斯拉執行長Elon Musk的官方推特，都沒有特斯拉即將推出手機的相關資訊。

（二）查核中心諮詢採訪ePrice比價王總編輯Jason表示，目前特斯拉推出手機只是傳聞，並無可信的證據佐證。

（三）以關鍵字「Tesla phone」檢索《路透社》、《CNN》、《美聯社》、《BBC》、《法新社》等媒體，也沒有看到特斯拉即將推出手機的相關報導。

特斯拉要發布手機並非新謠言，根據《法新社》曾在2021年12月與2023年6月發布與特斯拉手機有關的事實查核報導。兩則報導分別指出，微博、抖音、臉書的簡體中文社群，以及臉書的緬甸社群，都曾流傳特斯拉要出智慧型手機。但《法新社》指出，這些照片有些是經過造假，有些是電腦生成，都不是真實照片，特斯拉官方也未曾發布有關智慧型手機的相關聲明。

此外，印度查核組織《Factly》與《YouTurn》也分別在2022年12月與2023年1月針對網傳「Telsa Model π」進行查證後均指出，此為2021年以來熱傳的謠言，並不是事實。

網傳「特斯拉手機即將上市」，傳言的流傳狀況為何？

查核記者透過以圖反搜、關鍵字檢索等方式，追溯傳言源頭，發現有關特斯拉手機的傳言，最早是流傳於2021年。根據《今周刊》的報導以及3C電子資訊網站「電腦王阿達」的文章，設計師 Antonio De Rosa 在2021年初，設計了一款特斯拉概念手機「Tesla Model π」，引起諸多想像與討論。

經檢索，可找到設計師 Antonio De Rosa 在他營運的網站與YouTube頻道都發布了「Tesla Model π」的設計概念說明。

查核記者也在檢索的過程中發現，在 Antonio De Rosa 發布「Tesla Model π」之後，也有許多影片創作者與設計師跟進，以「Tesla Model π」為概念，設計、發想特斯拉的手機，並製作介紹影片。同一時間，有許多網路文章與YouTube影片在缺乏背景脈絡的情況下討論「Tesla Model π」的性能，易使人誤以為這是特斯拉將上市的手機。

綜合上述，網傳特斯拉手機「Tesla Model π」，源於設計師Antonio De Rosa在2021年的發想，後續有許多影片創作者或設計師跟進設計；而網傳手機照片經查出自YouTube頻道「Ts Designer」，並不是特斯拉官方推出的手機。

經檢索特斯拉各個官方平台以及多個外媒報導，都無「Tesla Model π」或特斯拉手機的相關資訊。

補充資料

網傳手機圖片出處為何？

網傳照片的手機桌布中有「Ts Designer」字樣，以此為線索可找到一個名為「Ts Designer」的YouTube頻道，其在2021年11月14日發布了一支影片「Tesla Pi 5G First Look Trailer Concept」，其中的手機設計與網傳照片相同。

經檢視此頻道，發布的內容均為各式手機介紹。該頻道共發布過三次有關特斯拉手機的影片，除了2021年11月的影片，該頻道也曾於2022年2月發布「Tesla Phone 5G」、2022年4月發布另一支「Tesla Pi」，這兩支2022年所發布手機影片，其中的手機照片外觀相同，但均與它在2021年發布的「Tesla Pi 5G」不同。可知此頻道發布的特斯拉手機影片並不是真實訊息。
